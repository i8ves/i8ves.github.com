# win 10/11 使用旧版的图片查看器

::: tip 前言
微软为 Win10 和 Win11 的开发了全新的照片应用，功能很强大，但也有一些旧版的优点没有继承过来，比如它还不支持循环查看图片，有时候单纯地想浏览图片选图就很不方便。
在Win10中还能通过「打开方式」菜单来修改为「Windows照片查看器」，到了Win11，微软直接把 Windows照片查看器屏蔽掉了，但是实际上此组件还是存在系统里的。
只需要将以下注册表代码复制粘贴进记事本，保存为 EnableViewer.reg
，文件类型选择「所有文件」，然后双击这个文件，同意导入注册表，现在就可以在打开方式的程序列表中看到旧版Windows照片查看器了（可能需要点「更多程序」或者「选择其他应用」才能看到）。

:::
